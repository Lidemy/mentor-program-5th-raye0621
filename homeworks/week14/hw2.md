寫給自己看的到底做了甚麼：

### 第一步從 AWS 購買主機

- EC2
- Ubuntu
- port 80 port 443
- 金鑰檔案

獲得 IPv4

### 設定 LAMP 環境

- 用金鑰匙 SSH 遠端主機
- Tasksel
- lamp-server

### 設定 phpmyadmin (設定資料庫)

- 下載 phpmyadmin 設定帳密
- 這樣就成功獲得遠端資料庫

### 放自己的檔案

- /var/www/html 這個位址
- 可以用 git clone (要先改權限)
- 本周是用 FileZilla 連線並直接放檔案進去
